# 🏥 Healthcare Excel Dashboard Project

## 📌 Overview

This project is an Excel-based dashboard created to analyze healthcare data and generate useful insights. It focuses on understanding patient trends, hospital performance, and treatment patterns.

---

## 🎯 Objective

* Analyze patient data across hospitals
* Understand trends in admissions and treatments
* Identify key performance indicators
* Support data-driven decision-making

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Slicers

---

## 📊 Dashboard Features

* Patient trends over time
* Age group and gender analysis
* Hospital performance comparison
* Insurance and admission analysis

---

## 📈 Key Insights

* Patient data shows variation across years
* Major hospitals contribute higher patient volume
* Certain age groups dominate healthcare usage
* Data helps in improving hospital decisions

---

## 📁 Files Included

* Excel Dashboard (.xlsx)
* Dashboard Screenshot

---

## 🧠 Conclusion

This project shows how Excel can be used for data analysis and visualization to support business decisions.
